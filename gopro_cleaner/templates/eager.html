    Clean footage

    How to use (click to collapse)
    1. Scan your folder — wait for snapshots to build.
    2. Filmstrip: use ← → to jump. If 4+ thumbnails in a row show no work, that block is garbage.
    3. Mark start I at the first useful frame. Fine-tune with , . or ±3s.
    4. Mark end O at the last useful frame.
    5. Trim T — saves NAME-1.MP4 beside the raw file. Repeat for more clips.
    6. Next file N — removes the raw file (if clips were saved) and continues.

    Shortcuts

    Navigate snapshots
    ← → Previous / next snapshot
    , . Fine-tune ±1 second (at trim point)
    Mark & trim
    I Mark clip start
    O Mark clip end
    T Trim & save marked clip
    File actions
    N Finish file & go to next
    K Keep entire file as one clip
    S Skip file (no changes)

    Label tasks

    How to use (click to collapse)
    1. Switch here after cleaning. Scan trimmed clips on the same folder.
    2. Watch each NAME-1.MP4 clip quickly.
    3. Pick a task from the list (or add a new one).
    4. Press N — clip moves to Labeled/<task>/.

    Shortcuts

    Labeling
    N Move to task & next clip
